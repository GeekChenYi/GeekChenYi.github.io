它动态的构建选择和使用属性或变量的值作为任意字符串

```less
p{
    color: ~"green"
}
```

编译后

```css
p{
    color: green
}
```

凡是里边写~"some_text"将被编译为CSS代码后，显示为some_text

创建var.less文件

```less
@namespace: geek;
```

创建style.less文件

```less
@prefix-cls: ~"@{namespace}-collapse-container"
```

lessc编译style.less后的文件style.css

```css
@prefix-cls: geek-collapse-container;
```

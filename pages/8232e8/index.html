前端开发中Git的最佳实践

# 一、前端开发中Git的最佳实践

在前端开发中，一般使用Visual Studio Code作为开发工具，VS Code已经帮我们集成了非常强大的图形化git操作界面，可以非常方便的完成日常工作中基本的任务。

其次借助VS Code强大的插件来进一步追踪代码，降低错误率。

  • Git History可以借助此插件追踪每行代码的提交记录，提交时间，提交人,避免自己背锅。
  • GitLens-Git supercharged此插件，不用多说，直接安装就完事了。

借助VS Code的插件和git的一些命令，基本就能互相弥补，很好的完成日常的开发任务了。
